Starting the application

The application starts with the following panel:



  • Play button: gets you access to the ghost busters' game. If you want to get a discount, you'll need to play it. You can find the instructions in the menu bar. Once finished, it will send you back to the start panel.
  • Book button: gets you access to the list of castles available to book.
  • Play and book button: at first it lets you play and when finished it gets you to the list of castles.
