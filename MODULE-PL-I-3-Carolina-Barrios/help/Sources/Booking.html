Booking a castle

The process of booking a castle starts when clicking on one of the castles in the list (see 'search castles'). Then a window similar to the following image would be open.



All the castle's information is on the top. Then you should fill the information about the number of people, the arrival date, the number of nights and the number of rooms you'll book.

Once you finish filling the data, click on next. A new panel would be shown for you to confirm that all the information is correct.



In adittion, you can click on 'Apply discount' to apply the discount if you have one. If you had not inserted your id already, a dialog would be displayed for you to input it. A discount can only be used once.

If you find some of the information is incorrect, press on 'Cancel' to go back to the last panel.

Once you press on 'Confirm', the following panel would be shown for you to insert your data and finish the booking process.

If you press on 'Confirm', you would back to the castle's information panel.
